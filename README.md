<h1 align="center">熊爸助手 - 家长交流论坛</h1>
<h2 align="center"> 基于Spring+Spring MVC+Mybatis(Maven方式构建)</h2>

## 说明
    熊爸助手专用家长交流论坛，采用SSM框架+Maven构建。其中包含数据库创建sql文件。
    本项目基于开源论坛项目Genesis修改。

## 如何使用
```aidl
$ git clone 项目git地址

$ cd Genesis

$ mvn clean compile

$ mvn clean package

$ mvn clean install

$ mvn jetty:run
 
 http://localhost:8080
```
